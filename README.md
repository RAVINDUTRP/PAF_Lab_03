## Installation
To get started with this project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/lab3.git
   cd lab3
   ```
2. Build the project using Maven:
   ```bash
   ./mvnw clean install
   ```
3. Run the application:
   ```bash
   ./mvnw spring-boot:run
   ```
